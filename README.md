# Responsive Landing Page 🚀

A modern and responsive landing page built using Next.js, TypeScript, and Tailwind CSS. This project focuses on clean UI design, smooth user interactions, and mobile-first responsiveness.

## 🔥 Features

- Responsive design for all devices 📱💻
- Sticky navigation bar with scroll effect
- Smooth scrolling and hover animations
- Clean and modern UI/UX
- Optimized performance using Next.js

## 🛠️ Tech Stack

- Next.js
- TypeScript
- Tailwind CSS

## 📂 Project Structure

- Components-based architecture
- Reusable UI components
- Clean folder structure

npm install
npm run dev
